# Final Price Calculator (GitHub Pages)

A simple, pretty HTML/CSS/JS calculator that resets on every page load.

## Formula
Final Price = brutto / ((100 + VAT)/100) * ((100 - discount)/100)

## How to host on GitHub Pages
1. Create a new GitHub repo (e.g. `price-calculator`)
2. Upload everything from this folder to the repo root (keep the `assets/` folder)
3. Go to **Settings → Pages**
4. **Deploy from a branch** → `main` → `/root`
5. Your site will be available at:
   `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Customization
- Replace `assets/waldin-bg.png` with any image you like.
- Adjust background scale in `assets/styles.css`:
  `background-size: 220px;`
